# web-assembly-learning

#### wasm-bindgen

Используем плагин wasm-bindgen для генерации bridge-функций из Rust в JS и из JS в Rust. Это позволит нативно обращаться к функциям WASM, обмениваясь более привычным набором структур данных, а не просто числовыми типами.

Функции, для которых должны быть созданы brdige-функции, помечаются Rust-атрибутом `#[wasm-bindgen]`.

Собрать проект:
```bash
wasm-pack build --target web
```
